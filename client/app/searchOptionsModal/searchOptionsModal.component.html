Search Options
You can activate some extra filter via the search bar
Here is the formatting of the filter !option=filter, an exemple : !stat_res=poison
You need to validate your filter by pressing "Enter"

stat_res


For the stat resistance you can filter on either status, element and type, it will only show when the stat resistance is at least at 1 (for the max level/awake/...)
- fire, ice, wind, earth, lightning, water, light, dark
- slash, magic, strike, missile, pierce
- poison, blind, sleep, silence, paralyze, confusion, petrify, toad, charm, slow, stop, immobilize, disable, berserk, doom, frostbite, stun

buff


For every buff possible with skill (active, passive, ...), like the stat the value of the skill need to be at least at 1
There is more option here you can filter on skill which give killer, inflict status ailment, add resistance to status ailment, buff type attack resistance or buff type attack damage
- fire_killer, ice_killer, wind_killer, earth_killer, lightning_killer, water_killer, light_killer, dark_killer, human_killer, netherbeast_killer, beast_killer, demon_killer, dragon_killer, plant_killer, bird_killer, insect_killer, aquatic_killer, machine_killer, fairy_killer, undead_killer, stone_killer, metal_killer, arcana_killer, fennes_killer, float_killer, males_killer
- fire_res, ice_res, wind_res, earth_res, lightning_res, water_res, light_res, dark_res
- slash_res, magic_res, strike_res, missile_res, pierce_res
- poison_res, blind_res, sleep_res, silence_res, paralyze_res, confusion_res, petrify_res, toad_res, charm_res, slow_res, stop_res, immobilize_res, disable_res, berserk_res, doom_res, frostbite_res, stun_res
- fire_atk, ice_atk, wind_atk, earth_atk, lightning_atk, water_atk, light_atk, dark_atk
- slash_atk, magic_atk, strike_atk, missile_atk, pierce_atk
- poison_atk, blind_atk, sleep_atk, silence_atk, paralyze_atk, confusion_atk, petrify_atk, toad_atk, charm_atk, slow_atk, stop_atk, immobilize_atk, disable_atk, berserk_atk, doom_atk, frostbite_atk, stun_atk
- evade
- reraise
- ignore_fatal (guts)
- dispel, mass_dispel

break


This time it's skill which apply negative value, like reduce the resistance to some element. - fire_res, ice_res, wind_res, earth_res, lightning_res, water_res, light_res, dark_res
- slash_res, magic_res, strike_res, missile_res, pierce_res
- poison_res, blind_res, sleep_res, silence_res, paralyze_res, confusion_res, petrify_res, toad_res, charm_res, slow_res, stop_res, immobilize_res, disable_res, berserk_res, doom_res, frostbite_res, stun_res

- slash_atk, magic_atk, strike_atk, missile_atk, pierce_atk

skill_name


Pretty easy you just have to type the skill you want to search (only work with english skill name on global or japanese for jp version)
