Search Options
You can activate some extra filter via the search bar
Here is the formatting of the filter !option=filter, an exemple : !stat_res=poison
You need to validate your filter by pressing "Enter"

stat_res


For the stat resistance you can filter on either status, element and type, it will only show when the stat resistance is at least at 1 (for the max level/awake/...)

- fire, ice, wind, earth, lightning, water, light, dark
- slash, magic, strike, missile, pierce
- poison, blind, sleep, silence, paralyze, confusion, petrify, toad, charm, slow, stop, immobilize, disable, berserk, doom, frostbite, stun, curse

buff or break


For every buff or break possible with skill (active, passive, ...), like the stat, the value of the skill need to be at least at 1 for !buff and negative value will be shown with !break
There is more option here you can filter on skill which give killer, inflict status ailment, add resistance to status ailment, buff type attack resistance or buff type attack damage

- {{ formattedBuffType.type | lowercase }} : {{ formattedBuffType.formattedEffect }}

skill_name


Pretty easy you just have to type the skill you want to search (only work with english skill name on global or japanese for jp version)
